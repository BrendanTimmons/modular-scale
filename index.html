Typographic Modular Scale

By Brendan Timmons ― 30th of July, 2017.

When starting a design there are many considerations. Typography being the cornerstone of any good design can have huge effects on the overall feel. Choosing the typeface is only one step in the process. The sizes at which one composes their type can change the mood completely. Modular scale is a tool which helps typographers compose with harmonious rhythm.

Defining a Scale

During the sixteenth century, European typographers devised a set of common sizes which remained nearly unchanged for 400 years. This scale became known as the classic typographic scale with a scaling ratio of two. Classic musical scales also share the same ratio where on a piano each C note vibrates twice as fast as the prior C note.

A modular scale, like a musical scale, is a prearranged set of harmonious proportions. ― The Elements of Typographic Style, Robert Bringhurst 1992

A scale is made up by three things: ratio, number of steps, and baseline. If we consider the web, then choosing base of 18px using the classic scale produces for us:

9 a 18 a 36 a 72 a 144 a
